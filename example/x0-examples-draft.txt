## Example "Screen Overlay"

x0 Base System Object

- `sysObjButton.js` **Button** opens *x0-overlay-mode* on click
- Overlay Screen contains `sysObjTabContainer.js` **TabContainer**

Setup description

   - TabContainer opened in Overlay Mode on Button click
   - Overlay mode == Open Screen containing new Objects above existing Screen
   - Underlaying Screen Data stays exact the same when Overlay Screen gets closed
   - Overlay Screen acts like a Window in an Operating System
